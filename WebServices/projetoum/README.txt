trata-se de uma aplicação que criou dua tabelas:
uma de departamentos e outros de usuários.
A de usuários possui todos os dados requeridos pela questão.

Tais dados podem ser acessados pelo localhost através do postman,
e podem extrair dados da planilha ou salvar novos dados.

Como requerido pela questão, os dados podem ser recebidos
em XML se for o que foi escolhido dentro do Postman.

O projeto foi gerado através de Biblioteca Spring para Java.
